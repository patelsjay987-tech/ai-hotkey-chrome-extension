# AI Hotkey Chrome Extension

A Chrome extension that brings AI to your fingertips with **hotkeys**.  
Select text and instantly get summaries, deep searches, pros/cons, bullet points, tone changes, or chat — all without typing prompts.

---

## Features
- **Hotkey-based AI actions**: No need to type prompts.
- **Multiple AI tools**: Summary, Deep Search, Pros/Cons, Bullet Points, Tone Change, Quick Chat.
- **Fast responses**: Works on any selected text.
- **Safe design**: Users enter their own Groq API key — no secrets stored in the code.

---

## How to Run
1. Clone the repo.
2. Load as an **unpacked Chrome extension**:
   - Go to `chrome://extensions/`
   - Enable **Developer Mode**
   - Click **Load unpacked** → select this folder
3. On first use, enter your own **Groq API key**
4. Press alt+z to activate and see all the AI functions 
5. Select text and use hotkeys to interact with the AI
